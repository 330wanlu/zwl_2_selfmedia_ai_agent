## 3. 本阶段代码改动摘要

### 3.1 任务取消

- `POST /api/v1/tasks/{id}/cancel`
  - 未完成任务 → `status=cancelled` / `current_stage=cancelled`
  - 已完成或已取消 → `409`
  - 调用 `workflow_service.cancel_background_run` 取消进行中的后台 `ainvoke`
- `_run_graph`：若任务已被取消，异常路径不再覆盖成 `failed`
- 决策 API 原有校验：`task.status != "running"` → `409`（取消后自然生效）
- 前端：详情页「取消任务」+ 列表「已取消」标签

### 3.2 Prompt 打磨要点

| 文件 | 强化点 |
| --- | --- |
| `prompts/topic.py` | 数字钩子 / 痛点反差硬性要求；角度差异；人群年龄+肤质+场景 |
| `prompts/content.py` | 开头钩子、场景对比、互动结尾、改写时保留历史意见 |
| `prompts/storyboard.py` | 封面冲击力；中文 ≤6 字；禁止满屏字/英文乱码/真品牌 logo |

## 7. 遇到的问题与解决

### 问题 1：上一会话做到一半未写文档

- **现象**：取消 / Prompt / `test_stage4.py` 与全量自测已完成，但 `开发记录-阶段4.md` 未落盘。
- **处理**：本轮复核自测结果，补写本文档并再次冒烟。

### 问题 2：库里残留 `running` 挂起任务

- **现象**：品鉴/联调过程中有任务停在 `waiting_topic_selection` / `waiting_image_review`，列表显得「卡住」。
- **处理**：调用取消 API 清理；运营侧也可在详情页点「取消任务」。

### 问题 3：全量 `all` 耗时长、耗额度

- **现象**：`revise5` + `redraw3` + `e2e` 会多次调文本与生图。
- **处理**：脚本支持分子命令；日常回归可只跑 `review` + `cancel`；全量留给正式验收。

### 问题 4：控制台中文乱码

- **现象**：Windows PowerShell 打印中文方向可能乱码。
- **处理**：脚本把完整 UTF-8 写入 `test_stage4.log` / `stage4_review.json`；控制台用 ASCII 摘要。

### 问题 5：LangSmith 未开

- **说明**：清单为「可选」。`langsmith` 已在依赖中，但未配置 `LANGSMITH_TRACING`。
- **后续**：需要看逐步耗时/I-O 时再开，非本阶段阻塞项。
